# No-Brainer-Junction-2025
Greatest entry

example_data.xlsx excel sheet's data is made up and just for example purposes and not completely realistic

The source code files and their purposes:

load_data.py
- fetches data from example_data.xlsx excel file and changes it to array format

array_data_to_dictionary.py 
- takes data from load_data and changes to dictionary type format for easier use of data

data_to_graphs.py
- takes data and from dictionaries and math.py, then generates graphs from them 

math.py
- takes data from array_data_to_dictionary.py and uses functions, matrixes and algorithms to calculate different types of parameters such as means, standard deviations and overall effect parameters.
